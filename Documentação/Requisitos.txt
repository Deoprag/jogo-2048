Requisitos Funcionais:

RF01 - O software possibilitará várias telas de jogo

RF02 - O software deverá realizar a mescla de células iguais de acordo com a rodada

RF03 - O software deverá disponibilizar a quantidade de pontos que o jogador acumula durante a partida

RF04 - O software deverá permitir a escolha de dificuldade: fácil, normal e difícil

RF05 - O software possuirá um ranking com as melhores pontuações dos jogadores

RF06 - O software deverá permitir ao jogador salvar o jogo caso ele queira jogar em outro momento

RF07 - O software deverá permitir ao jogador se cadastrar na tela inicial

RF08 - O jogador ganhará a partida ao atingir certa quantidade de pontos

RF09 - O software deverá gerar uma nova célula com valor de 2 ou 4 a cada rodada


Requsitos Não Funcionais:

RNF 1.1 - O software possuirá instruções do jogo na tela de jogo

RNF 1.2 - O software deverá exibir uma tela de GAME OVER ao final da partida


RNF 09.1 - O software deverá exibir uma tela com a opção de aumentar a dificuldade ao se ganhar o jogo nos modos fácil e médio
